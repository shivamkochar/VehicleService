# VehicleService
A scheduling application for an auto garage.<br>
The garage provides services for cars, trucks and vans.<br>
The garage offers the following services to all vehicles:<br>
•	Tire Change<br>
•	Oil Change<br>
•	Diagnosis<br>
•	Engine Tune-up<br>
•	Brake replacement<br>
